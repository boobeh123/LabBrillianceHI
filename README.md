## Description
A static web page for LabBrillianceHI, a local business on Oahu that curates lab-grown diamonds and offers them without markup

LabBrillianceHI webpage - https://labbrilliancehi.netlify.app/

## Features
* Static webpage deployed with Netlify
* WebP image format

## Optimizations
* Custom domain - Point LabBrillianceHI.com to webpage
* Responsive to mobile viewports
* Accessible to screen readers using Semantic HTML & ARIA
* Directly sell products with Shopify
